# Phishing Email Detection & Awareness Project

## Overview
This project analyzes phishing email samples to identify phishing indicators and raise user awareness about phishing attacks.

## Objective
The objective of this project is to understand how phishing emails work and how users can identify and avoid them.

## Email Samples
Three phishing email samples were analyzed in this project.

1. Account Verification Scam
2. Package Delivery Scam
3. Bank Update Scam

## Analysis Approach
- Collected phishing email samples
- Inspected sender email domains
- Analyzed suspicious links
- Identified phishing indicators
- Classified risk levels

## Risk Classification
All analyzed emails were classified as **High Risk – Phishing**.

## Prevention Guidelines
- Verify sender email addresses
- Avoid clicking unknown links
- Never share passwords or OTPs
- Report suspicious emails to the security team

## Analysis 1 — Suspicious URL

### VirusTotal Result

### URLScan Result
- Hosted in China
- Direct IP address used instead of domain — common phishing indicator

### Conclusion
URL flagged as suspicious based on:
- Direct IP usage instead of domain name
- China hosted server
